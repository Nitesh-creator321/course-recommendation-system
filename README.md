# Intelligent Course Recommendation System

This project is a web-based application that recommends courses to students based on their interests and current study level.

## Features
- User Registration and Login
- Interest-based Course Recommendation
- Course Enrollment
- View Enrolled Courses
- User Profile

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## How to Run the Project

1. Install XAMPP
2. Start Apache and MySQL
3. Place the project folder inside the `htdocs` directory
4. Import the database file into phpMyAdmin
5. Open the project in browser:

http://localhost/course_recommender
